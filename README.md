# Voting App
A real-time voting application built with HTML, CSS, JavaScript, and Firebase Realtime Database, enabling users to participate in polls and view instant results.

## Features
- Cast votes for options in a poll
- View live vote counts and results
- User-friendly interface with responsive design
- Data syncs across devices using Firebase Realtime Database

## Demo
https://sahilsharma2246.github.io/Voting/

## Installation
1. Clone the repository: `git clone https://github.com/sahilsharma2246/Voting.git`
2. Open `index.html` in your browser

## Usage
1. Select a voting option
2. Click "Vote" to cast your vote
3. View live results updated in real-time

## Tech Stack
- HTML
- CSS
- JavaScript
- Firebase Realtime Database
